# Sentiment Analyzer

A simple web app that analyzes the sentiment (positive / negative / neutral) of text you type in.

Runs entirely in the browser — no backend, no API keys, no installs needed.

## Files

- `index.html` — page structure
- `style.css` — styling
- `script.js` — sentiment analysis logic
